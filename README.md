## How do I write a blog?

-   Install [Ruby](https://www.ruby-lang.org/en/);

-   install [gems](http://rubygems.org/);

-   install [Jekyll](http://jekyllrb.com/docs/installation/)

Having the requirements prepared, build it with jekyll:


```bash
$ jekyll serve
```
